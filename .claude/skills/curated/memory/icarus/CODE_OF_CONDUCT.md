# Code of Conduct

## Standard

Be respectful. Be constructive. Assume good intent.

This project is small and focused. We don't need a long document to say: treat people well, focus on the work, and help each other build something useful.

## Scope

This applies to all project spaces: issues, pull requests, discussions, and any communication about this project.

## Enforcement

If someone is being disruptive, the maintainers will address it directly. Repeated issues result in removal from the project.

## Contact

Open an issue or reach out to the maintainers directly.
